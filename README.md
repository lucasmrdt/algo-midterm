# Algo midterm

## Usage

It is hosted on [https://algo-midterm.netlify.app/](https://algo-midterm.netlify.app/)

## Installation (With docker)

```
docker-compose build
docker-compose up
```


## Installation (Without docker)

### Front

```bash
cd front
corepack enable
yarn install
yarn start
```

### Back

```bash
pip install -r requirements.txt
python3 run-back.py
```
